## About

I am a researcher at Kyushu Institute of Technology, building software for imaging experiments, optical sensing systems, and reproducible research workflows. My work usually sits close to real hardware and real data: cameras, optics, signal processing pipelines, and evaluation code that needs to be trusted by other researchers.

- Currently focused on optical imaging, image enhancement, and computer vision for sensing applications
- Comfortable moving between experiment design, data processing, algorithm implementation, and visualization
- Interested in robust research code: clear scripts, documented assumptions, and results that can be re-run
- Working across Korean, English, and Japanese research environments

## Current Focus

| Area | What I work on |
